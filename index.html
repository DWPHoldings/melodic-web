Melodic

Build responsive, mobile-first projects with Inspire's front-end design system for the Web.

Melodic is Inspire's toolkit for developing with HTML, CSS, and JavaScript. Consider it a living style guide. This package consists of Sass variables and mixins, a responsive grid system, extensive prebuilt components, and JavaScript sprinkles built on jQuery to add interactivity.

Installation

Include Melodic's source Sass and JavaScript files via npm, Yarn, or Rubygems. Package managed installs don't include documentation, but do include our build system and readme.
